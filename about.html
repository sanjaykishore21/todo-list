About This Project

This To-Do List application is a simple and user-friendly tool designed to help users manage their daily tasks efficiently. It allows users to add, track, and complete tasks in an organized way.

✨ Features

  • Add new tasks easily
  • Mark tasks as completed
  • Clean and responsive design
  • Dark mode with glowing UI

🎯 Purpose

The main goal of this project is to improve productivity and help users stay organized. It also serves as a practice project to enhance skills in HTML, CSS, and JavaScript.

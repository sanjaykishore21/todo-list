# 🌟 To-Do List Web App

A stylish and responsive To-Do List web app with dark mode and glowing UI, built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ✅ Add and manage tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks
- 🌙 Dark mode with glowing UI
- 🎨 Clean and modern design
- ✨ Smooth animations

---
